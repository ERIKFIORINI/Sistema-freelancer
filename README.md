# Devorium

## Resumo do trabalho

Sistema de gerenciamento de freelancers desenvolvido utilizando PostgreSQL.

O sistema permite:

- cadastro de clientes
- cadastro de freelancers
- gerenciamento de habilidades
- criação de projetos
- contratos entre clientes e freelancers
- avaliações de desempenho
- consultas e relatórios SQL
